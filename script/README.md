
# SOARE SCRIPT

## Minifying SOARE Code

**See [Recommended tools](../README.md#1-soare-minifier)**

To minify your SOARE code, use the online tool at <https://anto.infos.st/SoareMinifier/>

1. Navigate to the minifier website
2. Upload your SOARE code
3. Click the minify button
4. Download the minified output

## Importing SOARE Scripts

To import a SOARE script into your project:

```soare
loadimport "path/to/script.soare"
```

Ensure the script path is relative to your current SOARE instance location.
